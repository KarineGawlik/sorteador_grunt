# Sorteador Grunt

Random number generator developed with JavaScript, HTML and Less.

🔗 Live project: https://sorteadorgruntkarine.vercel.app/

## Technologies

- HTML
- Less
- JavaScript
- Grunt
- Vercel

## Purpose

This project was created to practice JavaScript logic, user input handling, random number generation and task automation with Grunt.

## Features

- Maximum number input
- Random number generation
- Result display on screen
- Project automation with Grunt
- Deployed application with Vercel
